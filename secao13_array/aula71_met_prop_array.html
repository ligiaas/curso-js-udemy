Arrays

Métodos e Propriedades

legth

retorna o total de registros do array

every()

executa uma função que retorna um booleano se o todos os ele do array são de determinado valor

some()

executa uma função que retorna um booleano se ao menos um ele do array é de determinado valor

filter()

executa uma função que retorna os elementos que correspondem a determinada condição

forEach()

faz um loop dentro do array sem precisar criar uma variável incrementadora p/ percorrer o array, sem precisar usar o loop for/while.A cada índice do meu array eu vou executar uma função.

indexOf()

executa uma função que informa o índice de determinado ele no array, contanto a partir da esquerda.

lastIndexOf()

executa uma função que informa o índice de determinado ele no array, contanto a partir da direita.

map()

faz uma interação e alteração em cada um dos el do array

concat()

Serve pra juntar arrays, concatenar arrays

join()

converte uma array em string, podendo passar um caracter separado como parâmetro.

toString()

converte uma array em string, como caracter separador a vírgula.

push()

serve para acrescentar el no array

pop()

remove o último item do array e devolve o item removido pra quem o chamou

shift()

igual ao pop() mas ao invés de remover o último, ele remove o primeiro item do array

unshift()

igual ao push() mas ao invés de acrescentar no final, ele acrescenta no início do array

reduce()

serve pra pegar um array e retornar um único valor, ele executa qq tipo de expressão mas deve-se passar os parâmetros anterior e atual, vc pode passar tbm o index e o próprio array como parâmetros. Serve tbm p/ concatenar strings em um array

reverse()

Apenas inverte a ordem dos elementos em um array, é um método destrutivo, altera o array.

slice()

retorna um pedaço de um array

splice()

ele inclui, exclui, inclui no meio do array
