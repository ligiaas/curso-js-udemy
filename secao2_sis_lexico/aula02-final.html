Sistema Léxico


  • Case-sensitive: diferencia letras maiúsculas e minúsculas.
  • Comentários: // de uma linha e /* */ múltiplas linhas
  • Declarações: cada linha de código é uma declaração. Vc pode colocar mais de uma declaração na mesma linha, basta colocar ; no final de cada declaração.
  • Literais: é cada valor que eu posso atribuir a cada variável, sendo string, num, boolean, array e obj.Vc tbm declara objetos literais com {}.Com array é com [].
  • Escape: para escapar a aspas do js (ser interpretado como texto) é só colocar /.Ex: "o "/HTML/" é legal."
  • Variáveis e Constantes: para criar variáveis vc pode usar "var" e "let". Para criar constantes vc usa "const" é pra valores imutáveis, q não sofre variação.
  • Null, undefined, NaN e Infinity: Undefined é quando eu não atribuo valor algum, Null é quando eu atribuo valor nulo, NaN(Not a Number) quando vc tenta fazer uma operação matemática mas um dos valores não é calculável.Ex: alert(20 * "ola").Infinity é quando o número é muito grande pro js mostrar.
  • Objeto global (window, global) e this: window é o objeto raiz no browser, ex: alert é uma função do window, do browser, ele não funciona no banco de dados. Quando vc vai tratar do banco de dados vc usa o global. This é quem representa o contexto atual.
