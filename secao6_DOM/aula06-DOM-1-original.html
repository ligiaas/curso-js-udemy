Document Obejct Model

Sempre que o JS precisar pegar um elemento do HTML ele usará os recursos abaixos. Cada tag html se torna um objeto no js com props e métodos e a estrutura html vai determinar a hierarquia entre esses objetos.

  • getElementById: Seleciona o elemento html através do id.
  • getElementByTagName: Seleciona uma coleção de objetos(nodelist) que constam com a tagName informada. Sempre que usar ess opção é preciso indicar o índice(index) a ser alterado dentro de um array mesmo que seja uma coleção de 1 objeto.
  • getElementByClassName: Seleciona uma coleção de objetos(nodelist). Sempre que usar ess opção é preciso indicar o índice(index) a ser alterado dentro de um array mesmo que seja uma coleção de 1 objeto.
  • querySelector: Seleciona apenas o 1º objeto, com a mesma sintaxe do CSS. Através de '#id', '.class', 'tag' e etc.
  • querySelectorAll: Seleciona uma coleção de objeto podendo ser uma coleção de 1, com a mesma sintaxe do CSS. Através de '#id', '.class', 'tag' e etc.
