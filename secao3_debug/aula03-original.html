Lógica de programação com Javascript


Para colocar um breakpoint no código, na aba Sources é só clicar em cima da linha desejada.Pra excluir é a mesma coisa.

Para prosseguir com o debugger: F10 = vai pra próxima linha, F8 = vai pro próximo breakpoint,

  • F8 = resume script execution => continua a execução
  • F10 = step over next function call => passe pra próxima chamada de função
  • F11 = step into next function call => entra dentro da próxima função
  • F11 = step out of current function => sai da função atual

Debugger: Dentro de uma função escreva "debugger;" = força uma parada no código pra ter interação e inspeção com o código.

Console.log mostra uma mensagem qualquer.

Pra saber o tipo de um dado de uma variável digita no console: "typeof nomevar".Pra converter uma variavel pra numero: "var = ParseInt(var)"

Ao finalizar a inspeção, excluir os consoles.log() e os debuggers; NÃO ESQUECER!!
